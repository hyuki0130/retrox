# Game Assets Setup Guide

This guide explains how to download and install game sprites from Kenney.nl.

## Quick Setup

1. Download the asset packs from the links below
2. Extract and copy the required files to the corresponding folders
3. Rebuild the app

## Required Assets

### 1. Shooter Game (`sprites/shooter/`)

**Source**: [Space Shooter Redux](https://kenney.nl/assets/space-shooter-redux)

| File | Description |
|------|-------------|
| `playerShip1_blue.png` | Player spaceship |
| `enemyRed1.png` | Enemy type 1 |
| `enemyRed2.png` | Enemy type 2 |
| `enemyRed3.png` | Enemy type 3 |
| `laserBlue01.png` | Player bullet |
| `laserRed01.png` | Enemy bullet (optional) |
